Practica de Iconos

Como Insertar iconos de Google

Para insertar los iconos utilizaremos la siguiente etiqueta en el head <link rel="stylesheet" href="https://fonts.googleapis.com/icon?family=Material+Icons">
Todos los iconos de google se encuentran disponible en la siguiente URL
Todos se pueden modificar desde la propia pagina web, para que nos queden con los estilos deseados y el diseño deseado.

Ejemplos de iconos

Para modificar el estilo del icono podemos usar style=" " en la etiqueta donde invocamos el icono
Este es el icono por defecto paid <span class="material-icons">paid></span>
Este es el icono en verde paid <span class="material-icons" style="color:green">paid</span>
Este es el icono mas grande paid <span class="material-icons" style="font-size: 32px;">paid</span>
Este es el icono mas grande y en verde paid <span class="material-icons" style="font-size: 32px;color:green;">paid</span>

En este caso hemos usado el icono llamado "paid", pero podemos introducir distintos.
paid paid home home delete delete

Ejemplos de Iconos

Para modificar el estilo del icono podemos usar style=" " en la etiqueta donde invocamos el icono
Este es un icono por defecto <em class="glyphicon glyphicon-download-alt"></em>
Este es el icono en verde <em class="glyphicon glyphicon-download-alt " style="color:green;"></em>
Este es el icono mas grande <em class="glyphicon glyphicon-download-alt " style="font-size: 32px;"></em>
Este es el icono mas grande y en verde <em class="glyphicon glyphicon-download-alt" style="font-size: 32px;color:green;"></em>
En este caso hemos usado el icono de descarga alternativa, pero podemos usar mas de 250 iconos distintos.
